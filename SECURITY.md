# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in this extension, please do **not** open a public issue.

Instead, please report it via [Method, e.g., email or private GH security advisory].

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Security Features

- **Manifest V3**: Built on the latest secure extension platform.
- **Content Security Policy (CSP)**: Strict CSP to prevent XSS.
- **Sanitization**: All user inputs and AI responses are sanitized before rendering.
